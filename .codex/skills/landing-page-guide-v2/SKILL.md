# Landing Page Guide V2

Use this skill when the user wants a landing page that should convert and feel memorable.

## Workflow

1. Define the product context:
- audience
- conversion goal
- brand personality
- what should feel memorable

2. Pick an aesthetic direction and commit:
- minimalist and refined
- editorial
- organic
- retro-futuristic
- maximalist
- brutalist

3. Combine that direction with the core landing-page structure:
- strong hero
- clear CTA
- product proof
- benefits
- testimonials
- FAQ
- closing CTA

4. Implement with design intention:
- typography with character
- deliberate color system
- meaningful motion
- non-generic background treatment
