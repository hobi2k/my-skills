# Landing Page Guide

Use this skill when the user wants a strong conventional landing page that follows proven conversion structure.

## Workflow

1. Clarify:
- product or service
- audience
- primary CTA
- stack preference, usually React or Next.js

2. Cover the essential sections:
- hero with clear value proposition
- primary CTA
- social proof
- media or product demo
- benefits or features
- testimonials
- FAQ
- final CTA
- legal or contact footer

3. Implement in a clean component structure.
4. Verify SEO, responsiveness, and accessibility.
