# Design Prompt Generator V2

Use this skill when the deliverable is a high-quality prompt for another AI tool, not a finished UI implementation.

## Workflow

1. Identify the product context:
- domain
- target users
- main conversion goal
- brand personality

2. Build the prompt in seven layers:
- domain research
- user journey
- emotional direction
- brand identity and goals
- design system
- component specs
- micro-interactions

3. Make the prompt practical:
- include constraints
- specify success criteria
- name must-have sections and components
- call out what to avoid

4. Format the output so it can be pasted directly into another tool.

## Notes

- Prefer one strong creative direction over many weak options.
- If the user wants real code, switch to `frontend-design` or a landing-page skill instead.
