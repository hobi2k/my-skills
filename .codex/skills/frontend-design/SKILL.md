# Frontend Design

Use this skill when design quality is a core part of the task, not just functional completeness.

## Workflow

1. Understand the product context:
- purpose
- audience
- technical stack
- accessibility and responsiveness requirements

2. Commit to a clear visual direction before coding:
- minimalist
- editorial
- brutalist
- organic
- retro-futuristic
- luxury
- playful

3. Implement with deliberate choices:
- distinctive typography
- cohesive color system
- meaningful motion
- layered backgrounds or textures
- non-generic layout composition

4. Verify the result:
- works on desktop and mobile
- accessible interactions
- no visual placeholders or half-finished polish

## Notes

- Avoid generic AI defaults such as bland layout grids, safe purple gradients, and interchangeable typography.
- If the project already has a strong design system, respect it instead of forcing novelty.
