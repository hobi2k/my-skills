# Midjourney Card News Backgrounds

Use this skill to produce prompt text for image-generation tools.

## Workflow

1. Identify:
- subject
- brand or mood
- preferred palette
- whether text overlay readability matters

2. Choose a style family:
- business or tech
- finance
- health or wellness
- education
- lifestyle

3. Generate several prompt variants with:
- scene description
- style keywords
- palette
- atmosphere
- `--ar 1:1`

4. Explain which prompt is safest for text overlays and which is more experimental.

## Resources

- `topics_reference.md`

## Notes

- Keep prompts background-friendly; avoid central objects that will fight with overlay text.
